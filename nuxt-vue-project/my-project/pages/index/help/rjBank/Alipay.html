【入金指引】支付宝

【大陆网银汇款操作步骤】

第一步：获取收款银行账户
第二步：购汇（将人民币兑换成港币）
第三步：跨境汇款（将港币汇至收款银行账户）
第四步：通知嘉银查收款项
支付宝汇款操作的具体步骤：

1. 获取收款银行账户

登录嘉银app，依次点击选择「我的」）-「业务办理」-「存入资金」，即可获取和查看您的专属收款银行账户。

2. 购汇并汇款

（1）打开支付宝APP，依次点击“首页”、“更多”→“资金往来”、“上银汇款”

（2）点击“立即汇款”，进入汇款页面，填写收款人信息、收款人银行账户、汇款信息

（3）购汇并汇款

3. 发起汇款通知

登录嘉银app，依次点击选择「我的」-「业务办理」-「存入资金」,点击“下一步”，上传汇款凭证。

4. 查看历史记录

登录嘉银app，依次点击选择「我的」-「业务办理」-「资金记录」，可查看历史存入记录。

5. 到账时间

发起汇款通知后，预计到账时间是1-4个工作日。
跨境汇款经国内银行受理后，会通过该银行的中转行逐次传递，最终到达收款行指定账户。
在中转过程中，汇款路径不固定，因此无法准确计算出到账的时间。
